# AI Mystic - 综合命理测试系统

一个高端精美的 AI 综合命理测试网页，结合八字、星座、塔罗牌等多种元素，为用户提供个性化的命运解读。

## ✨ 特点

- 🎨 **高端设计** - 精美的视觉效果，酷炫的动效
- 🤖 **AI 驱动** - 基于 MiniMax API 的智能分析
- 🔮 **综合测试** - 融合八字、星座、塔罗牌等多种命理元素
- 📱 **响应式设计** - 完美适配各种设备
- 💰 **商业模式** - 支持付费订阅

## 🚀 快速开始

### 1. 安装依赖

```bash
cd backend
npm install
```

### 2. 配置环境变量

编辑 `backend/.env` 文件，添加你的 MiniMax API Key：

```bash
MINIMAX_API_KEY=你的API密钥
```

### 3. 启动服务器

```bash
cd backend
npm start
```

### 4. 访问网站

打开浏览器访问：`http://localhost:3000`
